AllergyAssist

Allergy Assist was a web application create at Code Day for my first competition ever. We, (Adam Chung, Rahul Gudivada, Aiden Kiefer, and myself) wanted to make it easier for parents to find restaurants that their child could eat at. So this website would let parents quickly check popular restaurants for certain allergies in the food items. We ended up winning the Best Overall prize with this website and have decided to start developing it to put on the market.
